Overview

This design system establishes the visual foundation for Truist's Commercial Banking platform, ensuring consistency, accessibility, and professional brand representation across all user interfaces.

The system is built on the principles of clarity, accessibility, and scalability, providing a comprehensive set of guidelines and components that maintain brand integrity while delivering exceptional user experiences.

Multi-Modal Communication

Status and feedback information is communicated through three channels for maximum accessibility:

  • Color coding - Immediate visual recognition
  • Iconographic - Universal symbol understanding
  • Textual - Clear descriptions for screen readers

/* Design Tokens - CSS Custom Properties */ :root { /* Brand Colors */ --truist-purple: #663399; --truist-purple-hover: #553388; --truist-purple-light: #f3f0ff; /* Text Colors */ --text-primary: #1a1a1a; --text-secondary: #4a5568; --text-muted: #6b7280; /* Status Colors */ --status-success: #16a34a; --status-warning: #f59e0b; --status-error: #ef4444; --status-info: #3b82f6; /* Spacing Scale */ --space-xs: 0.25rem; /* 4px */ --space-sm: 0.5rem; /* 8px */ --space-md: 1rem; /* 16px */ --space-lg: 1.5rem; /* 24px */ --space-xl: 2rem; /* 32px */ }

Implementation Guidelines

This design system ensures consistent, accessible, and maintainable user interfaces across the entire Truist Commercial Banking platform. All components are built with scalability and reusability in mind.

Key Principles

  • Consistency: Use design tokens and predefined components
  • Accessibility: WCAG AA compliance in all implementations
  • Scalability: Component-based architecture supports growth
  • Performance: Optimized CSS and minimal dependencies
  • Brand Integrity: Authentic Truist visual identity
